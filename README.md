# 如何使用

```bash
npm i -g @bfchain/devkit-chaindata-torrent-creater
```

> 可以使用交互式的方式回答问题, 以填充参数

```bash
chaindata-torrent-creater
```

> 或者直接传入参数

```bash
chaindata-torrent-creater --bsaeurl "BASE_URL" --input "INPUT_FOLDER" --output "OUTPUT"
```
